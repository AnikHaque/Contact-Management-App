# Terminal-Based Contact Management App

## Features

- Add a contact (up to 2 total)
- View saved contacts
- Simple text-based menu
- Input/output via terminal only
- Uses only:
  - Basic variables
  - `echo`, `fgets(STDIN)`
  - `if`, `else`, `switch`, `while`
  - **No arrays, loops like `for`, JSON, or file system**
